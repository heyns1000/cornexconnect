# Sample Data for Testing

This directory contains sample CSV files for testing the CornexConnect™ v2.6 import functionality.

## Files

- **skus.csv** - 31 EPS cornice product SKUs with various styles
- **customers.csv** - 10 sample customer records from South African cities

## Usage

1. Start the CornexConnect™ application
2. Navigate to the "Import Data" section
3. Drag and drop these CSV files to import sample data
4. The platform will validate and insert the data using Zod + Drizzle upsert

## SKU Categories

The sample SKUs include various architectural styles:
- Classic, Modern, Contemporary
- Victorian, Georgian, Federal
- Art Deco, Mid-Century, Industrial
- Mediterranean, Tuscan, Spanish
- Gothic, Baroque, Colonial
- And more...

All products are EPS (Expanded Polystyrene) cornices in various sizes (80mm - 220mm)
with pricing in South African Rand (ZAR).
